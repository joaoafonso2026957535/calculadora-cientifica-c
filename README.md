# calculadora-cientifica-c

Projeto Final - Laboratório de Programação

## Objetivo

Desenvolver uma calculadora científica em linguagem C utilizando conceitos estudados na disciplina, como funções, estruturas condicionais, vetores e structs. O sistema foi projetado para oferecer uma experiência simples e intuitiva ao usuário, permitindo a realização de operações matemáticas básicas e científicas sem a necessidade de conhecer as fórmulas utilizadas.

## Estrutura do Projeto

1. Bibliotecas
   // Importação das bibliotecas necessárias para entrada de dados, exibição de informações, manipulação de texto e operações matemáticas.

2. Structs
   // Definição da estrutura utilizada para armazenar o histórico das operações realizadas.

3. Variáveis Globais
   // Declaração das variáveis compartilhadas entre as funções, como histórico e contador de operações.

4. Protótipos
   // Declaração antecipada das funções para que possam ser reconhecidas pelo compilador.

5. Main
   // Função principal responsável por iniciar o sistema e controlar o fluxo de execução.

    ├── Menu
    // Exibição das opções disponíveis para o usuário.

    ├── Switch Case
    // Direcionamento da execução para a funcionalidade escolhida pelo usuário.

    └── Controle de execução
    // Controle da continuidade ou encerramento do programa.

6. Funções
   // Implementação das funcionalidades da calculadora.

    ├── Operações Básicas
    // Funções de soma, subtração, multiplicação e divisão.

    ├── Operações Científicas
    // Funções de potência, raiz quadrada, seno, cosseno e tangente.

    ├── Histórico
    // Exibição das operações armazenadas durante a execução do sistema.

    └── Tratamento de erros (if/else)
